| TYPE     | 내용                                    |
|----------|---------------------------------------|
| feat     | 새로운 기능 추가                             |
| fix      | 버그 수정                                 |
| docs     | 문서 수정                                 |
| style    | 기능 수정이 없이 코드 스타일 수정                   |
| test     | 테스트 코드, 리팩토링 테스트 코드 추가                |
| refactor | 코드 리팩토링                               |
| build    | 빌드 파일 수정                              |
| chore    | 빌드 업무 수정, 패키지 매니저 수정 (gitignore 수정 등) |
| rename   | 파일 혹은 폴더명을 수정                         |
